# RoASt Configuration Wizard

The Configuration Wizard helps you create a `config.json` file for the RoASt (Robot Algorithm Synthesizer Tool).

For detailed instructions on how to use the wizard, including field descriptions and examples, please consult the **[RoASt Documentation](../docs/pages/overview.html)**.

**Live Config Wizard**: [Launch Configuration Wizard](https://luminousrobots.github.io/RoASt-Configuration-Wizard/)
